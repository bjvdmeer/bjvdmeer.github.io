Go Board Game using JavaFX

A Go board game for 2 players made in Eclipse IDE using Java SE 14 with the JavaFX 14 library for application development. The game has features such as single stone and group capture, an interactive and responsive board and a basic score system. Every move made by players is verified by a logic class that implements the rules of the Go board game. The main challenges in creating this game were figuring out how automatically create the board based on mathematical rules, how to make each intersection of lines interactive, and implementing single stone and group capture using recursive algorithms. For extensive documentation, please refer to the /doc folder in the GitHub repository.

Zapp home care mobile application

An android mobile application for home care workers, built using xamarin in C#/.NET in Visual Studio 2019. This app is coupled to a restful API made using the Symfony PHP framework. The home screen displays an overview of appointments in a 24 hour window for the user that is logged in. Each appointment can be clicked to go to an overview screen that displays the tasks to be completed, client information and the address location. Users can sign in/out for appointments and completed appointments are automatically removed. Additionally, any appointments in which changes occur since the previous database update are marked in red in the overview. Data is exchanged between the MySQL database and the app in JSON, which is deserialized into C#/.NET using Data Transfer Objects (DTO).
